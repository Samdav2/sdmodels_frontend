## ğŸ¯ Technical Details

### Glassmorphism Formula
```css
background: gradient with transparency (40-70% opacity)
backdrop-filter: blur(40-60px)
border: 1-2px with 10-60% opacity
box-shadow: colored glow
```

### Glowing Border Formula
```css
border: 2px solid rgba(255,107,53,0.3-0.5)
box-shadow: 0 0 30-50px rgba(255,107,53,0.1-0.3)
```

### Color Palette
- **Primary Glow:** `rgba(255,107,53,X)` - Orange
- **Secondary Glow:** `rgba(214,40,40,X)` - Red
- **Background:** `slate-900/slate-950` with transparency
- **Text:** `white`, `gray-100`, `gray-200`, `gray-300`
- **Accents:** `orange-300`, `orange-400`, `orange-500`
